let、const和块级作用域

发表于 2016-09-03   |   分类于 ES6

es6新增了新的语法来声明变量，let和const，提供了块级作用域的概念，值得我们学习

1. let

es6新增了let和const来声明变量，与var不同的是，let和const不会有变量提升(预解释)的情况出现，也就不会有预解释的诸多问题，这点，我们要区别开来
并且，因为没有预解释，变量必须在声明后才能使用

  • let声明的变量只在当前代码块中有效

2. 块级作用域

- es5中只有全局作用域和函数作用域，块级作用域是没有的，在es6中，用大括号{}包裹的是一个块级作用域；
- 在es5中我们通过闭包来实现独立作用域，避免变量污染全局变量，现在用{}即可
- 在块级作用域中声明的变量，作用域外获取不到

3. const声明一个只读的常量。一旦声明，常量的值就不能改变。其他的特性跟let一样
